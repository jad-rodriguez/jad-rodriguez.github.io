Difference between Margin, Border, and Padding

In this section of my blog, we'll learn about the difference between Padding, Border, and Margin.

The illustration on the left pretty much sums it up. We can also refer to this as the CSS Box Model.

Imagine it as parts of a Box. Let's discuss each part!

  • Content Box: The area inside the box where we can see the content is displayed.
  • Padding Box: The area that sits around the content or element (yellow space in the illustration). It is important to emphasize that padding goes around all the four sides of the content. We can control the size of it using padding and related properties.
  • Border Box: The border box is as we know it, the border. It wraps content and any padding inside of it.
    We can also control the size of it by using border and related properties.
  • Margin Box: Last but not the least, the margin box, which is the outermost layer wrapping the content, padding, and border as the space between the box and other elements. You guessed it right, we can control the size of it using margin and related properties.

Can you see how this concept applies to this very page you are currently on?
